## 🚀 Como Executar o Projeto

Para clonar e rodar esta aplicação em sua máquina, você precisará do [Git](https://git-scm.com) e do [Node.js](https://nodejs.org/) instalados.

### 1. Clonar o Repositório
```bash
git clone [https://github.com/seu-usuario/seu-repositorio.git](https://github.com/seu-usuario/seu-repositorio.git)
cd seu-repositorio 
```

### 2. Instalar Dependências

```bash
# Se você usa npm
npm install

# Se você usa yarn
yarn

# Se você usa pnpm
pnpm install
```

### 3. Executar a Aplicação

```bash
# Se você usa npm
npm run dev

# Se você usa yarn
yarn dev

# Se você usa pnpm
pnpm dev
```
